## Теория принятия решений — образовательная платформа

Данное веб-приложение предназначено для изучения теории принятия решений. Оно ориентировано на студентов, исследователей и специалистов, которые стремятся повысить качество принятия решений в различных сферах деятельности. Платформа позволяет не только изучить теоретические аспекты, но и применить их на практике.

### Возможности
- Изучение 3 ключевых моделей теории принятия решений с подробными объяснениями.
- Возможность ввода пользовательских данных для каждой задачи с автоматическим расчётом оптимального решения.
- Интуитивно понятный веб-интерфейс, обеспечивающий удобное взаимодействие с платформой.

### Технологии
- **HTML5 / CSS3 / JS** — технологии для разработки фронтенда.
- **Figma** — инструмент для дизайна и прототипирования пользовательского интерфейса.
- **Git + GitHub** — система контроля версий для управления проектом.
- **Python** — язык программирования, используемый для реализации алгоритмов.
- **FastAPI** — веб-фреймворк для создания API с высокой производительностью.
- **Pydantic** — библиотека для валидации данных на серверной стороне.
- **Docker** — средство для изоляции среды разработки и деплоя.

### Как запустить проект локально
Запуск производится из главной папки проекта: TRPP_Project
```
docker-compose up -d --build 
```
